Bootcamp 2021: Pixel Run

My first programming project consisted of a small game where you move the character between three lanes and avoid enemies. It was made for the 2021 two-week Bootcamp program. This Processing project demonstrates my ability to make classes, for loops, integrate and use external libraries, and more! The background and assets were either coded by me in for loops, or I drew them on another program and put it into Processing, or I borrowed and credited resources from the internet. Animation was done by using framecounts and an external gif library. The clouds and slimes accelerate due to an acceleration variable. The clouds themselves are also randomly generated every time the game starts up. A score was made by using millis(); and only starts counting from when the game is started.

Check out the code on Github: https://arachellee.github.io/Alice-Khins-Bootcamp-Project/
